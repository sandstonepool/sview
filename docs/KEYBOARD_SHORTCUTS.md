# Keyboard Shortcuts

Complete reference for all keyboard shortcuts in sview.

## Global Shortcuts

Available in all views:

| Key | Action |
|-----|--------|
| `q` | Quit sview |
| `Esc` | Quit sview / Close current overlay |
| `r` | Refresh metrics immediately |

## Normal Mode (Main Dashboard)

| Key | Action |
|-----|--------|
| `?` | Toggle help overlay |
| `t` | Cycle to next color theme |
| `p` | Open peer list view |

## Multi-Node Navigation

Available when monitoring multiple nodes:

| Key | Action |
|-----|--------|
| `Tab` | Switch to next node |
| `Shift+Tab` | Switch to previous node |
| `→` | Switch to next node |
| `←` | Switch to previous node |
| `1` | Select node 1 |
| `2` | Select node 2 |
| `3` | Select node 3 |
| `4` | Select node 4 |
| `5` | Select node 5 |
| `6` | Select node 6 |
| `7` | Select node 7 |
| `8` | Select node 8 |
| `9` | Select node 9 |

## Peer List View

When peer list is open (`p`):

| Key | Action |
|-----|--------|
| `↑`, `k` | Move selection up |
| `↓`, `j` | Move selection down |
| `Enter` | View selected peer details |
| `→` | View selected peer details |
| `r` | Refresh peer list |
| `p` | Close peer list |
| `Esc` | Close peer list |

## Peer Detail View

When viewing peer details:

| Key | Action |
|-----|--------|
| `Backspace` | Return to peer list |
| `←` | Return to peer list |
| `Esc` | Close to main dashboard |

## Help Overlay

When help is shown (`?`):

| Key | Action |
|-----|--------|
| Any key | Close help |

## Vim-Style Navigation

For users familiar with Vim:

| Vim Key | Standard Key | Action |
|---------|--------------|--------|
| `j` | `↓` | Move down |
| `k` | `↑` | Move up |
